## 如何使用vue

* 离线版本：可以在官网中下载相关js文件即可
* 在线版本：通过在head标签中头部的script标签使用如下方式即可：
  `<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>`
